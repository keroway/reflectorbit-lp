## タイトル / キャッチコピー

- タイトル: **REFLECTORBIT**
- 由来: _Reflect_（反射） + _Orbit_（軌道）
- キャッチコピー案:
  - 「弾くな、撃ち返せ。」
  - 「←/→ で回転、Space/Shift で軌道半径——3 キーで操る軌道反射アクション。」

## コンセプト（Concept セクション）

> 中心の Core を周回する Shield を操り、重力に引かれて加速する Meteor を反射。
> 弾き返したメテオは緑（味方色）に変わり、画面外の Spawner を攻撃・破壊する——
> 「守る」だけでなく「反射して攻撃する」軌道アクション。

オリジナリティの核:

- **3 キー操作**: ←/→ で回転、Space/Shift で軌道半径をコントロール
- 反射した弾は味方色（緑）に変色——「防御 → 攻撃」への転換が視覚でわかる
- **内周ほど高スコア**（最大 2.2×）。攻めるほど報われるリスクリワード設計
- 最適反射で発動する **PERFECT REFLECT** でボーナス＆連鎖コンボ
- **Meteor は Normal / Arc（曲がる）/ Fast（高速）/ Heavy（要 2 回反射）の 4 種**。発射予告の色と軌道で見分けて反射する
- **Stage / Endless / Daily の 3 モード**。Daily は日付ベースの共通シードで同日プレイヤー同士の「今日の記録」を競える
- 遊ぶほど **Shield カラー（EMBER / AURORA / SOLAR）がコスメティックにアンロック**——スコアに影響しない見た目の到達目標

## 操作方法（How to Play セクション）

| キー                      | 動作                                                     |
| ------------------------- | -------------------------------------------------------- |
| **← / →**（押しっぱなし） | Shield を反時計 / 時計回りに回転（手動操作・離すと停止） |
| **Space**（押しっぱなし） | 軌道半径を縮小（角速度が上がる）                         |
| **Shift**（押しっぱなし） | 軌道半径を拡大（角速度が下がる）                         |
| **P**                     | ポーズ / 再開                                            |
| **↑ / ↓**                 | タイトルメニューを選択（Space/Enter で決定）             |

> 物理メモ: 軌道は角運動量保存（ω = L / r²）。半径を縮めるほど速く回る。
> Meteor は重力で Core に向かって加速してくる。
>
> タイトルメニューの「MODE & DIFFICULTY」で ←/→ により **Stage / Endless / Daily** のモードを、
> ↑/↓ により **Easy / Normal / Hard** の難易度を切り替えられる。

Web 版はタッチ / ポインタ操作にも対応。

> 操作表の下には Space=縮小 / Shift=拡大 の対比を見せる図解動画（HyperFrames 生成、
> `public/videos/how-to-play.mp4`/`.webm`）を併設。動画と操作表で役割を重複させず、
> 操作表の情報だけで完結する構成にする（動画はあくまで視覚的補助）。
