# LP コピー素材

LP に載せる文言のドラフト。確定したものではなく、デザイン時のたたき台。

## タイトル / キャッチコピー

- タイトル: **REFLECTORBIT**
- 由来: *Reflect*（反射） + *Orbit*（軌道）
- キャッチコピー案:
  - 「弾くな、撃ち返せ。」
  - 「2 キーで操る、軌道反射アクション。」
  - 「Space と Shift。たった 2 つで宇宙を弾く。」

## コンセプト（Concept セクション）

> 中心の Core を周回する Shield を操り、重力に引かれて加速する Meteor を反射。
> 弾き返したメテオを画面外の Spawner にぶつけて破壊する——
> 「守る」だけでなく「反射して攻撃する」軌道アクション。

オリジナリティの核:

- **2 キー操作**（Space / Shift）で軌道半径を能動的にコントロール
- 「ただ防ぐ」ではなく「反射 → 攻撃」という能動的フック
- 連鎖（誘爆コンボ）による爽快感
- 可変軌道半径 × メテオ重力加速による「タイミングと距離感の二重読み合い」

## 操作方法（How to Play セクション）

| キー | 動作 |
|---|---|
| **Space**（押しっぱなし） | 軌道半径を縮小（角速度が上がる） |
| **Shift**（押しっぱなし） | 軌道半径を拡大（角速度が下がる） |
| **P** | ポーズ / 再開 |
| **↑ / ↓** | タイトルで難易度選択 |

> 物理メモ: 軌道は角運動量保存（ω = L / r²）。半径を縮めるほど速く回る。
> Meteor は重力で Core に向かって加速してくる。

Web 版はタッチ / ポインタ操作にも対応。
